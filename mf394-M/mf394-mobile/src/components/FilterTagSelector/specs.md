This interface allows uers to assign a category and some tags when adding or editing contacts. It is not a filter like other tag/category interfaces. It is for assignign values. 

There should be an "Edit Tags" link or icon that can be pressed to modify tags. 

Pressing a tagg will cause it to toggle selected or deselected. 

Default category is miscellaneous. 

This component will appear in the Add, Edit and Party workflows.
