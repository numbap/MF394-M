The Face Selector is used after face detection in the Add and Edit flows. The user an select one of the found headshots and it will be used as the selected image. 

User can click "Back" to return to the Add/Edit form without selecting an image.  
User can click Crop to manually crop the uploaded image instead of selecting one of the faces. 
Clicking on a face will add it to the image interface in the Add or Edit flow.
